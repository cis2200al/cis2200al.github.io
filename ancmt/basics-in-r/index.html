Dealing with Variables: Create, Rename, Recode & Merge

(Source: Tutorial by Klodian Dhana on datascienceplus.com)

1. Create a Dataset:

hospital <- c("New York", "California") patients <- c(150, 350) costs <- c(3.1, 2.5) df <- data.frame (hospital, patients, costs)

The data created is called df:

Hospital Patients Costs
New York 150 3.1
California 350 2.5

2. Create a New Variable:

A new variable called totcosts as shown below is created:

df$totcosts <- df$patients * df$costs

The new dataset will look have another column called "totcosts" with the two values that are multiplications of patients & costs.

3. Rename a Variable:

To rename the variable costs, for example, to costs_euro:

df$costs_euro <- df$costs

4. Delete a Variable:

To delete the variable costs:

df$costs <- NULL

5. Recode a Variable:

To recode variable patients:

df$patients <- ifelse(df$patients==150, 100, ifelse(df$patients==350, 300, NA))

This way, the patients column in the dataframe will be have the values of 100 and 300, instead of 150 and 350 as currently.

6. Merge Datasets:

To Add Columns: use merge(), which requires the datasets to have a common variable.

finaldt <- merge(dataset1, dataset2, by="id")

If datasets do not have a common variable, use the function cbind(), which requires that both datasets to have the same order.

finaldt <- cbind(dataset1, dataset2)

To Add Rows: use rbind(), which requires that datasets have exactly the same variable names and same number of variables.

finaldt <- rbind(dataset1, dataset2)
