Information Security Written Assignment

Due Date:

Wednesday, October 11, 11:59 pm

Objective:

Analyze the impacts of security breaches.

Requirements:

Team vs. Individual:

If you already are part of a group, complete this assignment with your team members, and turn in only one submission for your group.

If you still have not formed your group, complete this assignment on your own and submit the assignment as usual.

Details of Assignment:

Select a security breach that happened in the recent past and put down the name of that security breach next to your name and/or your group name in this spreadsheet on google drive.

You need to make sure that the security breach you select is different from the names that others have put down on the spreadsheet. So this means that you need to check the spreadsheet on google drive before you think of selecting any security breach, and that you need to put down your selection on the spreadsheet as soon as possible, so that no other team/individual can choose the same security breach as you.

If I find that any of you ends up writing about the same security breach, you will have to select another breach and do the assignment again.

Analyze the security breach that you selected, at a minimum, answering all of the following questions (you can add more analysis if you like):

  • 1 What are some important background information about the security breach? (Name of the affected organizations & entities, date, etc.)
  • 2 What are the consequences of the security breach? How did it affect the various stakeholders?
  • 3 What could the organization have done to avoid the security breach?

  • 4 What did the organization do, and should have done (but did not do), after the breach?

  • Cite any sources that you use (best to use APA style)

  • You need to provide full citations (not just mention the titles and authors' names in the text), regardless of which style you choose.
  • Font: Georgia
  • Font size: range from 10 pt to 12 pt
  • Spacing: Single Space
  • Word Count: Between 250-500 words
  • Submission: Please send the word document to the email address: athluong@gmail.com; with subject "Week 5 Assignment".
  • You need to send your submission to this email address, not my other email, in order to be graded. Thanks.
